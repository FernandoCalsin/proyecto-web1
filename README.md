## Breve escripción
Travelia es un sitio web de turismo cuyo objetivo es centralizar información relevante para los visitantes de distintos destinos turísticos.  La plataforma permite conocer la gastronomía típica, la ubicación, información general del lugar y las opciones de transporte en buses.  Además, ofrece la posibilidad de reservar visitas, facilitando la planificación del viaje.  Travelia busca mejorar la experiencia del usuario brindando información clara y herramientas prácticas en un solo entorno web.
